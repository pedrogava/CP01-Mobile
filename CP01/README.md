## Para criar o projeto

```sh
npx create-expo-app@latest AppNome --template blank-typescript
```

## Acesse a pasta

```sh
cd NomedoApp
```

## Build do App

```sh
npx expo start
```

## Android:

```sh
npx react-native run-android
```

## iOS (necessário macOS e Xcode):

```sh
npx react-native run-ios
```
